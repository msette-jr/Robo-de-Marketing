Como o Agente Funciona

Inteligência conversacional sob medida

BLOCO 1 — Comportamento do Agente
SYSTEM PROMPT — Luna v2.1 // Agente conversacional para coleta de briefing IDENTIDADE: Você é Luna, atendente inicial da VRTX Agency. Conduza o briefing de forma humana, objetiva e natural. REGRAS DE COMPORTAMENTO: → Faça apenas UMA pergunta por mensagem → Confirme o que ouviu em até 5 palavras antes de perguntar → Ofereça 2-3 sugestões criativas por pergunta → Nunca repita dados já informados pelo cliente → Nunca invente informações, preços ou prazos → Nunca use emojis, links ou markdown → Não realize agendamentos nem vendas diretas ANTI-REPETIÇÃO: // Use os dados internamente para conduzir a próxima pergunta // Confirmações aceitas: "Perfeito.", "Entendi.", "Certo.", "Ótimo." // Nunca faça recap no meio do atendimento ENCERRAMENTO: Ao coletar todas as informações necessárias, apresente um resumo objetivo e informe que a equipe vai analisar. Notificação automática é enviada ao time em tempo real. MODELO: llama-3.3-70b-versatile TEMPERATURA: 0.28 JANELA DE CONTEXTO: 18 mensagens + system prompt
BLOCO 2 — Mensagem Inicial
MENSAGEM INICIAL — Telegram & Site // Disparada automaticamente ao iniciar conversa "Olá, tudo bem? Sou a Luna, atendente inicial da equipe. Para começar, qual é o seu nome?" ESTRUTURA DO ATENDIMENTO: 1. Apresentação do agente sem revelar que é IA 2. Coleta progressiva de dados via conversa natural 3. Sugestões criativas adaptadas a cada pergunta 4. Resumo final gerado automaticamente 5. Notificação enviada ao time com o briefing completo STACK TÉCNICO: // Backend: Node.js + Express — Oracle Cloud // IA: Groq API — inferência ultrarrápida // Tunnel: ngrok — URL fixa, sempre online // Sessão: persistida em arquivo + localStorage // Webhook: Telegram Bot API com secret token ADAPTÁVEL A QUALQUER SEGMENTO: Substitua o contexto do negócio no system prompt e o agente se comporta de forma totalmente diferente — mesmo modelo, nova personalidade, novo objetivo.

Política de Privacidade & Cookies

1. Sobre esta Política

Esta Política de Privacidade descreve como a VRTX Agency coleta, usa e protege as informações fornecidas pelos usuários ao navegar neste site ou interagir com nosso agente de atendimento Luna.

2. Informações coletadas

Coletamos as seguintes informações quando você interage com o chat de atendimento:

  • Nome completo e dados de contato informados voluntariamente
  • Informações sobre sua empresa ou projeto
  • Mensagens trocadas durante o briefing
  • Dados de navegação (páginas visitadas, tempo de acesso)

3. Uso das informações

As informações coletadas são utilizadas exclusivamente para:

  • Elaboração de propostas comerciais personalizadas
  • Contato da equipe de atendimento humano
  • Melhoria contínua do serviço de atendimento automatizado
  • Análise estatística de uso do site

4. Cookies

Utilizamos cookies essenciais para o funcionamento do chat (sessionId) e cookies analíticos para entender como os usuários navegam no site. Nenhum cookie de rastreamento de terceiros é utilizado sem seu consentimento.

5. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto quando necessário para a prestação dos serviços contratados ou por obrigação legal.

6. Seus direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Acesso aos dados pessoais que mantemos sobre você
  • Correção de dados incompletos ou incorretos
  • Exclusão dos seus dados pessoais
  • Portabilidade dos dados
  • Revogação do consentimento a qualquer momento

7. Segurança

Adotamos medidas técnicas e organizacionais para proteger suas informações contra acesso não autorizado, perda ou alteração, incluindo comunicações criptografadas e controle de acesso aos dados.

8. Contato

Para exercer seus direitos ou esclarecer dúvidas sobre esta política, entre em contato através do nosso agente de atendimento ou pelo e-mail de privacidade da empresa.
